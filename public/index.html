What is audiomark?

audiomark allows audio snippets to be collected on your PC, laptop where they are saved locally in a PouchDB database.

supervisor_account

How do I share?

Once the clip is recorded you can print a QR code that allows someone else to listen to your recording.

grade

What's it for?

It's used by teachers to affix stickers to students' work instead of writing long-form comments on their work.

PouchDB allows data to be stored inside the web browser. Audio is stored as an attachment to a PouchDB JSON document. All of your audio snippets are stored in your web browser until you log out.

Cloudant is a NoSQL database-as-a-service running in the cloud. PouchDB sends its data to Cloudant where they are stored so that a user who visits a URL encoded in a QR code is delivered the audio file.

Cloudant Envoy is a microservice that sits between PouchDB and Cloudant. It ensures that each users' data is stored in a single Cloudant database but in a way that each user cannot overwrite each other's records.

MSR or MediaStreamRecorder is a JavaScript library that allows cross-browser audio and video recording with a simple API.

qrcode.js is a JavaScript library that encodes a string as a QR code on an HTML canvas control. The code can be read by any QR code reader app and decoded back into its original url.

MaterializeCSS is a front-end framework that allows mobile-friendly web pages to be created that follow Google's Material Design principles.
